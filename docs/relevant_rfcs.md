# Relevant RFCs
This is a list of the RFCs referenced during the development of RSMTPD:

* [RFC 5321: Simple Mail Transfer Protocol](https://tools.ietf.org/html/rfc5321)
* [RFC 3207: SMTP Service Extension for Secure SMTP over Transport Layer Security](https://tools.ietf.org/html/rfc3207)
* [RFC 7817: Updated Transport Layer Security (TLS) Server Identity Check Procedure for Email-Related Protocols](https://tools.ietf.org/html/rfc7817)
* [RFC 2920: SMTP Service Extension for Command Pipelining](https://tools.ietf.org/html/rfc2920)
* [RFC 6152: SMTP Service Extension for 8-bit MIME Transport](https://tools.ietf.org/html/rfc6152)
